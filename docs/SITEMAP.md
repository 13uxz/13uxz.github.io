# 13uxz.com — Sitemap & Wireframes

## Sitemap

Single-page site with anchor-linked sections:

```
13uxz.com/
├── #hero        — Artist name, tagline, CTA
├── #bio         — Photo + biography text
├── #music       — Beatport links + SoundCloud
├── #photos      — Performance photo grid
├── #contact     — Email contact form
└── footer       — Socials + copyright
```

## Navigation

Fixed top nav bar with section links:
`[13uxz]  ·  Bio  ·  Music  ·  Photos  ·  Contact`
